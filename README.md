# Gatsby shop

#### Application -> https://gatsbyshop.netlify.app

#### Before usage

```
nvm use
npm install
```

### start project

#### Dev

```
npm run start:dev
```

#### Prod

```
  npm run json-server // should work during build
  npm run build
  npm run start:prod
```

## Or run in docker

```
docker-compose -f docker-compose.dev.yaml up
```
